# 운영 체크리스트

## 매일 (운영일)
**아침 (-2h 출발 전)**
- [ ] 오늘 예약 명단 확인 (인원·시간·옵션)
- [ ] 파트너에게 인원·시간 최종 통보
- [ ] 기상·수상 컨디션 확인
- [ ] 픽업 일정·동선 확인
- [ ] 장비/구명조끼 사이즈 분포 점검

**도착 시**
- [ ] 체크인 (이름·인원·건강 상태)
- [ ] 안전 브리핑 (영문)
- [ ] 구명조끼 핏 확인

**활동 중**
- [ ] 그룹 동선·인원 수 수시 확인
- [ ] 사진/영상 촬영 (동의 받은 케이스)

**종료**
- [ ] 분실물 점검
- [ ] 리뷰 요청 메시지 발송
- [ ] CRM에 결과 기록 (Showed/이슈/리뷰 약속)
