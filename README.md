# Surfen

외국인 고객 대상 수상레포츠 송객 사업의 운영 허브.

## 사업 개요

- **모델**: 파트너 사업장(**서프앤 / Surfen**)에 외국인 고객을 송객하고 커미션을 받는 구조
- **담당 영역**: 마케팅 · 세일즈 · 현장 가이드/CS
- **타겟**: 한국 방문 외국인 관광객(영어권 중심)
- **파트너 특징**: 풀라인 수상레포츠 + **제트서프·플라이포일보드 전문** + **협회·대회 운영 권위** — 자세한 사항은 [`01_business/partner-info.md`](01_business/partner-info.md)
