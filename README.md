# Surfen

외국인 고객 대상 수상레포츠(빠지) 송객 사업의 운영 허브.

## 사업 개요

- **모델**: 파트너 사업장(친구의 빠지/수상레포츠)에 외국인 고객을 송객하고 커미션을 받는 구조
- **담당 영역**: 마케팅 · 세일즈 · 현장 가이드/CS
- **타겟**: 한국 방문 외국인 관광객(영어권 중심)

## 디렉토리 구조

| 폴더 | 용도 |
| --- | --- |
| `01_business/` | 사업 계획, 파트너십 조건, 수익 모델 |
| `02_marketing/` | 타겟 페르소나, 채널 전략, 콘텐츠, 에셋 |
| `03_sales/` | 가격 정책, 예약 플로우, 파이프라인, 영업 스크립트 |
| `04_cs/` | 영문 FAQ, 가이드 스크립트, 안전 브리핑, 트러블슈팅 |
| `05_operations/` | 일일/주간 운영 체크리스트, 작업 관리 |
| `docs/` | 기타 참고 문서, 회의록 |

## 시작 가이드

1. `01_business/overview.md` 에서 사업의 큰 그림과 파트너십 조건을 정리합니다.
2. `02_marketing/target-personas.md` 에서 외국인 고객 페르소나를 구체화합니다.
3. `03_sales/booking-flow.md` 에서 문의 → 예약 → 송객 흐름을 설계합니다.
4. `04_cs/faq-en.md`, `guide-script-en.md` 에서 영문 응대 자료를 준비합니다.

## 협업

- 모든 문서는 한국어로 작성하되, 고객 응대용 자료(`04_cs/`)는 영문으로 작성합니다.
- 변경사항은 의미 있는 단위로 커밋합니다.
