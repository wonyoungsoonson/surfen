# 파이프라인 / 예약 트래킹

> 초기엔 이 파일로 시작, 양이 늘면 구글시트/노션으로 이전.

## 컬럼 정의
| 컬럼 | 설명 |
| --- | --- |
| Date | 문의 일자 |
| Name | 고객 이름 |
| Country | 국적 |
| Channel | 유입 채널 (Insta, Klook, FB, Google, Referral …) |
| Party Size | 인원 |
| Product | 상품 |
| Stage | Inquiry / Quoted / Confirmed / Showed / No-show / Refunded |
| Booking Date | 예약 일자 |
| Amount (KRW) | 금액 |
| Commission (KRW) | 커미션 |
| Notes | 메모 |
