# "Free Stay Included" 포지셔닝 전략

> 사장님 정책 (₩100k+ 숙박 무료 / ₩200k+ 숙박+BBQ 무료)을 마케팅 핵심 메시지로 전환.
> 외국인 OTA·소셜·검색 전 채널에서 일관된 "Stay Included" 포지셔닝.

---

## 핵심 한 줄

**"Cheongpyeong's only water sports school where every package comes with a free lakeside stay."**

가평·청평 어떤 경쟁사도 외국인 OTA에서 "숙박 포함"을 내세우지 않음 → 첫 진입자가 카테고리 차지.

---

## 왜 이 메시지가 강한가

### 1. 외국인 OTA 검색 패턴과 정확히 매칭
Klook·Trazy에서 외국인이 검색하는 패턴:
- "things to do in Gapyeong" (액티비티)
- "weekend getaway from Seoul" (1박)
- "stay near Seoul" (숙박)
- "Korea lake house" (호수)

→ 우리는 **이 4개 검색어를 하나의 상품으로 묶음**. 외국인이 보통 4개 따로 예약하던 걸 1개로.

### 2. 가성비 인식 폭발
외국인 입장에서 Gapyeong 1박:
- 숙소 (글램핑/펜션) ₩100,000~150,000
- 액티비티 (워터파크) ₩30,000
- BBQ ₩50,000
- 합계 ₩180,000~230,000 (~$130~165)

우리:
- Fliteboard 90분 + 무료 1박 = ₩150,000 ($108)
- 웨이크서프 60분 + 무료 1박 + 무료 BBQ = ₩360,000 ($260)

→ **숙박·BBQ 포함하면 우리가 더 싸거나 비슷한데, 종목이 카테고리 위**.

### 3. 신뢰·프리미엄 시그널
"무료" 단어는 보통 싸구려·미끼 인식. 하지만:
- **명품 워터스포츠**(Fliteboard) + **무료 호수변 숙박** = 럭셔리 호텔의 "complimentary upgrade" 같은 느낌
- 메시지: "We don't charge you separately for the bed because the whole experience is one."

### 4. 콘텐츠 자산 무한 생성
숙박이 있으면 콘텐츠 축이 늘어남:
- 도착 (체크인, 룸 둘러보기)
- 액티비티
- BBQ (저녁 일몰)
- 숙박 (룸 안에서 야경)
- 아침 (호수 산책)
- 출발

→ 1박 콘텐츠 = 30초 영상 5~10편 분량. 데이트립 콘텐츠 대비 5배.

## 메시지 프레임 (사용 vs 금지)

### ✅ 사용
- "**Stay included**"
- "**Sleep where you surf**"
- "**Wakesurf today, wake by the lake tomorrow**"
- "**Lakeside cabin included**"
- "**Korea's only stay-included water sports**"
- "**One package. Activity + Stay + BBQ.**"

### ❌ 금지 (싸구려 인식 유발)
- "FREE!" (대문자 강조)
- "Buy X get Y FREE" (쿠폰 인식)
- "₩100,000 이상 사면 숙박 무료" (한국 사장님 식 카피)
- "Promotion: free stay" (한정 프로모션 같음)
- "지금만!" (urgency tactic)

### 핵심 원칙
**숙박을 "조건부 혜택"이 아니라 "상품 구성 요소"로**. 가격에 녹여서 묶음 상품으로 판매.

---

## OTA 상품 구조 — Bundle-First

**과거 구조 (조건부 혜택)**:
> "웨이크서프 60분 ₩360,000 — ₩200k 이상이라 숙박+BBQ 무료"

**새 구조 (Bundle-first)**:
> "Wakesurf & Lakeside Stay Package — ₩360,000
> 60-min wakesurf + lakeside cabin (1 night) + BBQ dinner + breakfast"

→ Klook 페이지에서 사진·설명·가격이 모두 "1박 패키지" 톤. 숙박이 옵션이 아니라 디폴트.

### 상품 매트릭스 (Bundle 적용)

| 상품명 | 가격 | 포함 |
| --- | --- | --- |
| **Lakeside Park Day + Stay** | ₩100,000 | 초보강습 2회 + 놀이기구 4종 + 1박 |
| **Fliteboard Discovery & Stay** | ₩150,000 | Fliteboard 90분 + 1박 |
| **Wakesurf Beginner & Stay** | ₩150,000 | 웨이크서프 초보 30분 + 1박 |
| **Wakesurf Premium & BBQ Stay** | ₩360,000 | 웨이크서프 60분 + BBQ 저녁 + 1박 + 아침 |
| **Wakesurf Half-Day & BBQ Stay** | ₩720,000 | 웨이크서프 120분 + BBQ 저녁 + 1박 + 아침 |
| **Jet Ski & BBQ Stay** | ₩400,000 | 젯트스키 60분 + BBQ 저녁 + 1박 + 아침 |
| **Luxury Yacht & Stay (group)** | ₩700,000/boat | 요트 60분 + BBQ 저녁 + 1박 (그룹) |
| **Nami Island Boat Tour & Stay** | ₩300,000~700,000 | 보트 투어 + BBQ + 1박 |

### 가격대 자동 인센티브 매칭
| 사장님 정책 | 우리 패키지 |
| --- | --- |
| ₩100k+ → 숙박 무료 | 모든 시그니처 패키지 (Fliteboard·웨이크 초보·콤보) |
| ₩200k+ → 숙박+BBQ 무료 | 웨이크서프 60분·120분·젯트스키·요트·럭셔리 보트 |
| 비수기 평일 40% 할인 | 별도 "Weekday Special" 상품 |

→ 모든 시그니처 상품이 자동으로 "Stay Included" 또는 "Stay + BBQ Included".

## 컴페티터 비교 카드 (OTA·블로그·콘텐츠 동시 활용)

| 항목 | 일반 가평 워터파크 (Trazy ₩26k 빠지 패스) | Surfen Wakesurf & Stay (₩360k) |
| --- | --- | --- |
| 액티비티 | 빠지·바나나 무제한 | 웨이크서프 60분 (프리미엄) |
| 영문 가이드 | 없음 | 있음 |
| 숙박 | 별도 (₩100~150k 추가) | **포함** |
| BBQ | 별도 | **포함** |
| 픽업 | 가평역 셔틀 | 청평역 픽업 (서울 더 가까움) |
| 총 비용 (1박 기준) | ₩26k + 숙박 ₩120k + BBQ ₩60k = **₩206k** | **₩360k** |
| 카테고리 | 가족 워터파크 | 프리미엄 액션 + 호수 리트리트 |

→ **비교하면 단일 가격 차이 같지만, 카테고리·경험이 완전히 다른 상품**. Trazy ₩26k 손님과 Surfen 손님은 다른 페르소나.

## 사장님께 확인 필요 (Bundle 진행 전)

Bundle 상품으로 OTA에 올리려면 숙박 품질이 부족하면 리뷰 망함.

- [ ] **객실 수** (Klook 등 OTA에서 인벤토리 관리 필요)
- [ ] **객실 타입** (글램핑? 컨테이너? 펜션?)
- [ ] **객실 사진 5장+** (OTA 페이지에 직접 사용)
- [ ] **객실 어메니티** (침구·수건·에어컨·와이파이·샤워)
- [ ] **수용 인원** (2인실 / 4인실 / 가족실)
- [ ] **체크인·체크아웃 시간**
- [ ] **객실 예약 슬롯** (한국 손님 vs 외국인 패키지 동시 예약 시 어떻게 조율?)
- [ ] **숙박 미사용 시 환불·할인 옵션** (외국인이 당일 귀가 원할 경우)

이게 확인 안 되면 Klook 등록 시 "Stay Included"를 약속만 하고 실제 객실은 부실할 위험 → 리뷰에서 1~2점 폭격 받음. **숙박 품질이 ★★★ 이상 확보되어야 이 전략 안전**.
