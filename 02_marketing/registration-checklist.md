# 채널 등록 체크리스트

> 서프앤(Surfen) 기본 셋팅. 우선순위 = 외국인 도달 × 등록 난이도 × 신뢰도 효과.

## 공통 준비물 (모든 채널 진입 전 1회 준비)
- [ ] **사업자등록증** (영문 표기명도 같이 — "Surfen" / "SURFEN")
- [ ] **사업 설명** 영문 1줄 / 50자 / 200자 / 1000자 4종
- [ ] **사진 30장 이상**
  - 액티비티(제트서프·플라이포일·웨이크·요트·빠지·바나나보트) 각 5장
  - 강사·고객·시설·로고
  - 가로·세로 두 비율 다 준비
- [ ] **영상 3개 이상**
  - 30초 하이라이트(인스타·OTA용)
  - 60~90초 액티비티별
  - 안전 브리핑(영문 자막)
- [ ] **가격표** (영문/한글)
- [ ] **환불·취소 정책** (영문/한글)
- [ ] **영문 FAQ** (`../04_cs/faq-en.md` 활용)
- [ ] **위치/픽업 정보** (Google Maps 좌표, 픽업 가능 거점 리스트)
- [ ] **사업자 명의 카드/계좌** (정산 수신용)
- [ ] **대표 이메일** (예: hello@surfen.kr — 도메인 확보 시)

## 등록 순서 (실행 워크플로우)

```
[Week 1]
- Google Business Profile 신청 (우편 인증 1~2주 대기 시작)
- Instagram 비즈니스 전환 + 링크트리
- TikTok·YouTube 채널 개설
- Naver Place 영문 보강

[Week 2]
- TripAdvisor 사업장 등록
- 공통 준비물 (사진·영상·영문 카피) 완성
- Klook Activity Provider 신청

[Week 3~4]
- Google Business 인증 완료 → 리스팅 최적화
- Klook 승인 대기 → 상품 페이지 본격 작성
- Founding Group(외국인 지인) 1차 체험 일정 잡기

[Week 5~8]
- Klook 정식 오픈 → 첫 5명 송객 → 리뷰 확보
- TripAdvisor 리뷰 5개 목표
- 다음 OTA(KKday, Viator) 신청

[Month 3+]
- Visit Korea 등 공식 인프라
- 협회·대회 자산을 PR로 연결
- 자체 도메인·랜딩 검토
```
