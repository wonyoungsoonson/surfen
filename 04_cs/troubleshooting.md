# 트러블슈팅 (운영 케이스)

> 자주 발생하는 상황과 대응 표준. 새 케이스가 발생하면 추가.

## 노쇼
1. 예약 시간 +10분 대기 후 메시지·전화 시도
2. 응답 없을 시 노쇼 처리 → 환불 정책 적용
3. CRM에 사유·시간 기록
4. (반복 노쇼 패턴이면) D-1 리마인드 강화

## 우천/풍랑 등 운항 불가
1. D-1 17:00 시점에 기상 예보 1차 확인
2. 운항 불가 판정 시 즉시 모든 예약자에게 영문 공지
3. 100% 환불 또는 일정 변경 선택지 제시
4. OTA는 OTA 채널 통해 별도 처리

## 현장 부상/응급
1. 활동 즉시 중지
2. 응급처치 → 119 또는 가까운 의료기관 안내
3. 파트너 매니저에게 즉시 보고
4. 사고 보고서 작성: 일시·인원·경위·조치
5. 후속 안내 (보험 청구, 사후 연락)

## 분실물
1. 현장 1차 수색
2. 발견 시 픽업·배송 옵션 제시
3. 미발견 시 1주일 보관·기록, 분실물 페이지 안내

## 클레임/부정 리뷰
1. 24시간 내 사과 + 사실 확인 답글
2. 보상이 필요한 경우 파트너와 협의 후 결정
3. 내부 회고: 같은 클레임 재발 방지 액션

## 결제/환불 분쟁
1. OTA 경유: OTA 정책 우선
2. 직접 결제: 영수증·통화 기록 확인 후 정책 적용
3. 챠지백 발생 시 파트너에게 즉시 공유
